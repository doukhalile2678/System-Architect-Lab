# System Architect Lab

Projet de Système d’Exploitation réalisé avec :

- Bash
- Docker
- Debian Linux
- Python
- MySQL

## Objectif

Le projet permet de :

- automatiser des tâches Linux avec Bash
- simuler des algorithmes d’ordonnancement CPU
- utiliser Docker et Docker Compose
- manipuler une base de données MySQL
